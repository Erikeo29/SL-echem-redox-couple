# CV & EIS — Redox Couple on Gold Electrode

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://echem-redox-couple.streamlit.app/)

Bilingual (FR/EN) Streamlit application for visualizing electrochemical simulations: cyclic voltammetry (CV) and electrochemical impedance spectroscopy (EIS) of the ferro/ferricyanide redox couple on a gold electrode.

## Studies

| Study | Method | Parameters |
|-------|--------|-----------|
| **1. CV** | FEM 2D (Firedrake) | k⁰, α, scan rate ν |
| **2. EIS** | Randles + CPE + Warburg | n, k⁰, D, c, Q₀ |
